# 91约拍Pro 生产就绪审计 V1.0

> 审计基线：`91约拍Pro 产品功能点清单 V1.0`
>
> 审计原则：页面存在、按钮可点击、静态数据可展示、前端本地存储可流转，均不视为业务功能完成。只有数据库、服务端接口、权限校验、状态机、异常处理、审计日志、测试和部署配置共同完成，才可标记为“可上线”。

## 1. 结论

当前仓库属于：

- RuoYi-Vue3-FastAPI 通用管理框架底座；
- UniApp 用户端 UI 原型；
- 部分本地存储模拟的约拍流程；
- 不具备真实交易上线条件。

按 MVP 第一阶段 18 项进行严格验收：

- 生产完成：0 项；
- 有 UI 或本地模拟：11 项；
- 基本缺失：7 项；
- 上线结论：禁止直接发布生产环境。

## 3. 当前可复用能力

以下能力可以继续作为生产项目底座使用，但必须补测试和业务集成：

- FastAPI + SQLAlchemy Async；
- MySQL / PostgreSQL 接入能力；
- Redis；
- OAuth2 / JWT；
- RBAC 菜单与按钮权限；
- 操作日志、登录日志、定时任务；
- 前后端传输加密；
- UniApp + Vue 3 + Vite；
- 通用管理后台。

## 4. 当前严重问题

### 4.1 静态数据冒充业务数据

以下模块存在大量写死数组、金额和状态：

- 摄影师、模特、作品、需求；
- 消息与聊天；
- 报价与订单；
- 钱包与收入；
- 档期；
- 创作者工作台；
- 作品交付与评价。

### 4.2 本地存储冒充服务端状态

本地存储只能用于草稿和非关键偏好，不能作为以下数据的事实来源：

- 订单；
- 支付状态；
- 报价；
- 报名；
- 交付验收；
- 评价；
- 退款；
- 钱包余额。

### 4.3 生产配置不合格

- 移动端接口地址硬编码；
- 应用名称仍包含模板名称；
- 官网和协议链接仍指向模板项目；
- 登录页默认填充管理员账号和密码；
- 无开发、测试、预发布、生产环境隔离；
- 无密钥管理和生产配置校验。

### 4.4 缺少交易级后端

仓库尚未形成以下核心领域：

```text
用户身份域
创作者与商家域
内容与作品域
约拍需求域
报名与邀请域
服务套餐域
档期域
会话与消息域
报价域
订单与履约域
支付、退款与结算域
交付与版权域
评价与信用域
审核、举报与风控域
活动、场地与营销域
```

### 4.5 缺少上线质量门禁

必须补齐：

- 后端单元测试和集成测试；
- 数据库迁移测试；
- API 契约测试；
- 支付回调和退款幂等测试；
- 小程序端核心流程 E2E；
- 权限越权测试；
- 上传安全测试；
- 性能与容量测试；
- 灰度、监控、告警、备份和回滚演练。

## 5. 正式上线必须满足的完成定义

每个业务功能必须同时具备：

1. 数据库模型和迁移脚本；
2. 服务端 API；
3. 鉴权和数据权限；
4. 参数校验；
5. 状态机和非法状态拦截；
6. 幂等、事务、并发控制；
7. 操作审计；
8. 前端真实接口接入；
9. 加载、空、失败、弱网和重试状态；
10. 自动化测试；
11. 管理后台能力；
12. 监控、告警和运营配置。

## 6. 推荐实施顺序

### P0-A：账号、身份与安全基础

- 微信登录、手机号验证码登录；
- 用户资料和多身份；
- 实名/职业/商家认证；
- 隐私、黑名单、设备和注销；
- 文件上传和内容审核基础服务。

### P0-B：约拍交易核心

- 创作者资料与服务套餐；
- 需求、报名、邀请；
- 档期和冲突锁；
- 聊天与报价；
- 订单状态机；
- 微信支付、退款和对账。

### P0-C：线下履约与交付

- 到场签到、开始和结束；
- 作品上传、选片、修改和验收；
- 双向评价、信用和投诉；
- 平台结算与提现。

### P0-D：管理后台与运营

- 认证审核；
- 内容审核；
- 订单、退款、结算；
- 举报、客服、风控；
- 首页运营位、城市、类目和协议配置。

## 7. 外部上线依赖

代码完成后，真实上线还必须准备：

- 微信小程序 AppID；
- 微信支付商户号、API v3 密钥和平台证书；
- 已备案 HTTPS API 域名；
- 短信服务；
- 对象存储和 CDN；
- 身份核验/人脸识别服务；
- 文本、图片和视频内容安全服务；
- 地图和定位服务；
- 推送/订阅消息模板；
- 隐私政策、用户协议、交易协议、退款规则和未成年人保护规则；
- ICP/公安备案及相应经营合规材料；
- 生产 MySQL、Redis、备份、监控和日志平台。

## 8. 当前验收规则

从本审计开始：

- 静态 UI 不标记完成；
- `uni.setStorageSync` 不得保存交易事实；
- 模拟支付不得标记为支付完成；
- 无服务端状态机不得标记订单完成；
- 无审核后台不得标记认证/内容审核完成；
- 无自动化测试和部署验证不得标记可上线。
